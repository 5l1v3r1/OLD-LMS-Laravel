# LMS-Laravel
[![Latest Stable Version](https://poser.pugx.org/lms-laravel/lms-laravel/v/stable)](https://packagist.org/packages/lms-laravel/lms-laravel) [![Total Downloads](https://poser.pugx.org/lms-laravel/lms-laravel/downloads)](https://packagist.org/packages/lms-laravel/lms-laravel) [![Latest Unstable Version](https://poser.pugx.org/lms-laravel/lms-laravel/v/unstable)](https://packagist.org/packages/lms-laravel/lms-laravel) [![License](https://poser.pugx.org/angelkurten/lms-laravel/license)](https://packagist.org/packages/angelkurten/lms-laravel)

LMS-laravel is a management system of educational content, want to facilitate the creation of a platform simple and intuitive

LMS-laravel is based as its name indicates in the framework laravel 5, and uses various packages created by other developers.
